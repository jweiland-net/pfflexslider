============
Users Manual
============

Target group: **Editors**

#. Select a page where you want to insert pfflexslider

#. Insert a new content element of type: `Insert Plugin`

#. On tab `Plugin` select `Pfflexslider` from selectbox.

#. Confirm reload

#. Save

Slider Record
-------------

#. Select a Storage Page

#. Create a new record of type `Slider`

#. Give it a title, an image and, if needed, a link.
